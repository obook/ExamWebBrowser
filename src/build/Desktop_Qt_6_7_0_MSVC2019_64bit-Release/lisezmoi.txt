Déploiement initial (rapide et grossier)
Fermer Qt Creator ;
Copier les élements suivant dans C:\Deployment\
La version release de MyApp.exe ;
Tous les fichiers .dll de C:\Qt\5.2.1\mingw48_32\bin\ \ ;
Tous les répertoires de C:\Qt\5.2.1\mingw48_32\plugins\ ;
(Si vous utilisez QML) Tous les répertoires de C:\Qt\5.2.1\mingw48_32\qml\.
Renommer C:\Qt en C:\QtHidden\ (Cela donne à votre PC un environnement propre, comme si Qt n’y était pas installé) ;
Tester C:\Deployment\MyApp.exe.
Si tout est OK : Renommer C:\QtHidden\ en C:\Qt
